# App da Carteira Hipotecaria e imobiliaria do Clube Militar

App desenvolvido com o intuito de gerenciar os documentos digitalizados.

## Tecnologias utilizadas:
- Node.js
- Express.js
- Bootstrap
- React.js
- Mysql
- Sequelize ORM
